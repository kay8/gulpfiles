# gulpfile

1: Install modules

```shell
npm install
```

2: Install Ruby and [scss-lint](https://github.com/brigade/scss-lint)

```shell
gem install scss_lint
```

3: Set up folder structure

.
├── assets
│   ├── scripts
│   └── styles
│       └── partials
├── dist
    ├── scripts
    ├── styles
    └── images
